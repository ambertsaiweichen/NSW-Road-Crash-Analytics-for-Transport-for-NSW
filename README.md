## 🔎 Overview
A data analytics and machine learning project developed for Transport for NSW to identify key factors contributing to road crashes and fatalities between 2017 and 2021.<br/>
The project applies **data preprocessing**, **exploratory data analysis**, **data visualisation**, **statistical modelling**, and **machine learning techniques** to uncover crash patterns and support evidence-based road safety recommendations.

## 📋 Scnenario
Road fatalities remain a significant public safety challenge across Australia.

Transport for NSW sought analytical insights from historical crash data to better understand:
- What factors contribute to severe crashes
- Which locations experience the highest crash frequency
- Whether crash severity can be predicted using environmental and road-related variables
- How future road safety campaigns and resources should be allocated

The objective was to support NSW's long-term road safety targets through data-driven decision making.

## 📊 Data Sets
NSW Road Crash Data (2017–2021)

Data Includes:<br/>
▪️ Crash severity<br/>
▪️ Speed limits<br/>
▪️ Weather conditions<br/>
▪️ Road classifications<br/>
▪️ Road alignment<br/>
▪️ Time and date of crashes<br/>
▪️ Fatalities and injuries<br/>
▪️ Geographic crash locations<br/>

Analysed over five years of historical crash records across New South Wales.

## 💡 Key Analytics Performed
### Data Preparation
- Removed duplicates and missing values
- Cleaned and transformed raw crash records
- Encoded categorical variables for machine learning
- Performed feature engineering and preprocessing workflows
### Exploratory Data Analysis
Developed visualisations to analyse:<br/>
- Top crash-prone towns and suburbs
- Fatalities by weather condition
- Monthly and yearly fatality trends
- Speed limit versus crash severity
- High-risk road locations
- Geographic crash density heatmaps across NSW
### Predictive Modelling
Built and evaluated:<br/>
- Linear Regression
- Logistic Regression
- Decision Tree Classification
Model performance was assessed using:<br/>
- R²
- RMSE
- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC and Precision-Recall Curves

## 📈 Key Findings
▪️ Western Sydney suburbs recorded some of the highest crash frequencies.<br/>
▪️ Most fatalities occurred during fine weather conditions.<br/>
▪️ High-speed roads showed greater fatality risks.<br/>
▪️ Crash severity could be predicted more effectively using classification models than linear regression approaches.<br/>
▪️ Geographic hotspots were concentrated within metropolitan Sydney regions.<br/>
